Minesweeper Tutorial

Geschichte

Most successful game ever
Minesweeper wurde ursprünglich von Microsoft für Windows 3.1 entwickelt und danach jeder Nachfolgeversion für den PC beigelegt; seit Windows 8 muss es als App aus dem Windows Store nachinstalliert werden. Das Spiel fehlt bei Windows Mobile bis zur Version Windows Mobile 2003 SE. Für Windows Phone wird in vielen Ländern über den integrierten Windows Phone Store eine kostenlose Xbox Live Minesweeper-App[1][2] angeboten. Mit Windows Vista wurde es vollständig überarbeitet und besitzt zum Beispiel die Möglichkeit, die Minen durch Blumen zu ersetzen[3]. Minesweeper war ebenfalls im 1999 erschienenen Mobiltelefon Siemens S25 integriert. Rekorde, die auf Windows beiliegenden Programmversionen erzielt wurden, werden für internationale und nationale Bestenlisten nicht akzeptiert, da die aktuelle Version dem Spieler die Möglichkeit gibt, auf einem Spielfeld mit den gleichen Minenpositionen mehrmals hintereinander zu spielen (Neustart) und vorhergehende Versionen nicht sicher gegen Cheats und Hacks sind. Für Wettkämpfe und Ranglisten finden dagegen ausschließlich die Programme „Minesweeper Arbiter“, „Minesweeper X“ und „Viennasweeper“ Anwendung. In Anlehnung an Minesweeper wurde das papierbasierte Spiel Beleuchtung entwickelt.

Tutorial Guide

Das Spiel kann in verschiedenen Schwierigkeitsstufen gespielt werden. Dabei unterscheidet sich nicht nur die Anzahl Minen, sondern auch die Spielfeldgrösse

Aufgedeckte Bereiche

Wenn ein Feld angeklickt wird, welches keine Nummer hat, werden automatisch alle anderen Felder auch aufgedeckt, bis wieder Felder kommen, die eine Zahl haben.

Minen entdecken

Die Eins in der Ecke gibt an, dass das Feld mit dem roten X, das Feld sein muss, unter dem die Mine ist. Ausserdem sind alle anderen Felder aufgedeckt, also muss es das sein.

Eine Flagge

Wenn sie sich sicher sind, dass sich unter dem Feld eine Mine befindet, tätigen sie rechtsklick und platzieren eine Flagge. Dies verhindert, dass sie versehentlich dieses Feld mit der linken Maustaste anklicken und sich die Position für die Mine zu merken.

FAQ

  • Das Spiel ist mir zu einfach, was kann ich tun, um es schwerer zu gestalten?
    Falls dir das Spiel zu einfach ist, kannst du einmal versuchen das Spiel ohne Flaggen zu spielen. Alternativ kannst du auch versuchen eine neue Bestzeit auzustellen.
